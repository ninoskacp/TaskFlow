# TaskFlow - Aplicación de Gestión de Tareas

Este proyecto es la entrega oficial para la evaluación del módulo **Programación Avanzada en JavaScript**.

## 🚀 Características Implementadas
* **Programación Orientada a Objetos (POO):** Clases `Tarea` y `GestorTareas` para modularizar la lógica.
* **Características ES6+:** Uso de funciones flecha, Template Literals, Destructuring y el operador Spread.
* **Asincronía Avanzada:** Notificaciones temporizadas con `setTimeout` y contador regresivo con `setInterval`.
* **Consumo de APIs y Persistencia:** Integración de peticiones HTTP mediante `fetch` (Async/Await) con respaldo en `localStorage` ante errores de red (bloques `try/catch`).
